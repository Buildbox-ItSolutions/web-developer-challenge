## Getting Started

Run in development server:

```bash
npm run dev
```

Run in prod:

```bash
npm run start:prod
# or
docker-compose up

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Libs

Produced in Next.js with:

- TypeScript
- Styled-components
- Fakerjs
