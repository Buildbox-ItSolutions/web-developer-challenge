## Sobre o Projeto

Projeto feito para o Desafio Técnico BuildBox.

## Iniciando

Primeiro, instalar as dependências do projeto:

```bash
    yarn
```

E por fim, execute o projeto:

```bash
    yarn dev
    or
    yarn build
    yarn start
```
